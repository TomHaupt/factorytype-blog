The Space Rⁿ and its Topology

The Distance Function in Rⁿ

Points in the space Rⁿ are represented as n-tuples of real numbers, denoted as (x₁, x₂, …, xₙ), where each xᵢ is a real number. The space Rⁿ is equipped with the standard Euclidean topology, which is defined using a distance function (or metric) that measures the distance between two points in Rⁿ. The distance function d: Rⁿ × Rⁿ → R is given by: $$d((x₁, x₂, …, xₙ), (y₁, y₂, …, yₙ)) = √((x₁ - y₁)² + (x₂ - y₂)² + … + (xₙ - yₙ)²)$$ This metric induces a topology on Rⁿ.
If the space is R², the distance between the points A and B is given by: d((0,0), (3,4)) = √((0-3)² + (0-4)²) = √(9 + 16) = √25 = 5
