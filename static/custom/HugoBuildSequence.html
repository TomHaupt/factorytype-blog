Hugo Build Sequence

  • 1. Load Configuration
    Reads config.toml, theme settings, taxonomies, and environment variables.
  • 2. Discover Content
    Scans content/ for Markdown files and parses front matter.
  • 3. Generate Taxonomies
    Builds tag/category indexes and list pages.
  • 4. Render Markdown
    Converts Markdown to HTML, applies shortcodes, syntax highlighting, and TOC.
  • 5. Resolve Layouts
    Matches content type to layout templates and applies theme overrides.
  • 6. Process Assets
    Copies static/ files, reads data/, and runs Hugo Pipes if used.
  • 7. Assemble Pages
    Combines content, layout, metadata, and partials.
  • 8. Generate Output
    Writes HTML, XML, RSS, sitemap, and other files to public/.
  • 9. Optional Minification
    Compresses output if --minify is used.
  • 10. Ready to Push
    Use git add, commit, and push to deploy via Netlify.
